Project Name:
ISO 15939 Measurement Process Simulator

Description:
This project is a Java Swing desktop application that simulates the 5 core steps of the ISO/IEC 15939 software measurement process standard.

Features:
- Profile step with validation
- Define step with scenario selection
- Plan step with metric tables
- Collect step with score calculation
- Analyse step with gap analysis and radar chart
- Step indicator using CardLayout wizard structure

Compilation Instructions:

1. Open the project folder in terminal.
2. Compile the project using:

javac -d out src\model\*.java src\view\*.java src\controller\*.java src\service\*.java src\repository\*.java src\Main.java

Run Instructions:
Run the application using:

java -cp out Main
